## 集成建议

### 前端实现
- 在**工具跳转页面**中使用此列表
- 通过小程序 API 调用链接：`wx.navigateToMiniProgram()`
- Web 链接使用：`window.open()` 或 `<a>` 标签

### 后端实现
- 将工具列表存储在数据库的 `apps` 表中
- 提供 `/api/tools` 或 `/api/apps` 端点获取列表
- 支持按分类筛选：`/api/tools?category=医疗知识库`

### 新增工具流程
1. 医学顾问委员会审核新工具
2. 更新本文档和数据库
3. 更新前端工具列表 UI
4. 发布版本更新
